### Features

- Multi-layer encryption (AES ‚Üí 3DES ‚Üí DES ‚Üí Blowfish)
- Support for symmetric encryption algorithms:
  - AES (CBC mode)
  - 3DES (Triple DES)
  - DES
  - Blowfish
- Dynamic key generation based on user parameters
- Data compression using zlib
- Base64 encoding for easy transmission
- Data serialization using marshal

### Installation

```

```

### Usage

#### CLI Mode

```bash
python marte.py
```

#### Usage in Code

```python

# Encryption

# Decryption

```
